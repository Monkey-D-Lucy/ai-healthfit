# HealthFit AI - AI-Powered Health & Fitness Platform

## Overview
HealthFit AI is a comprehensive health and fitness platform that leverages artificial intelligence to provide personalized health insights, track vital metrics, and offer intelligent recommendations.

## Features
- 📊 Real-time health metrics tracking
- 🤖 AI-powered health risk prediction
- 📅 Appointment scheduling with doctors
- ⌚ Virtual smart watch dashboard
- 🏋️ Exercise form analysis
- 💤 Sleep quality tracking
- ❤️ Heart rate monitoring
- 📈 Health trends visualization
- 🎨 Beautiful glassmorphism UI

## Tech Stack
- **Frontend**: React 18, TypeScript, Redux Toolkit, Tailwind CSS
- **Backend**: Flask, SQLAlchemy, JWT Authentication
- **Database**: MySQL 8
- **AI Service**: Flask, NumPy
